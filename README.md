CareerSim is a career discovery simulation platform that evaluates a user's
decision-making style through scenario-based tasks and maps it to career
compatibility.

## Problem Statement
Students often choose careers based on marks or peer pressure instead of
decision-making traits and real-world thinking patterns.

## Solution
CareerSim simulates real-life career scenarios and evaluates user choices using
a deterministic AI decision engine.

## Features
- Career-based simulations
- Decision tracking
- Compatibility scoring
- Strength & weakness analysis
- Glassmorphism UI with gradient design

## Tech Stack
Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Firebase Functions
- Express.js

## AI Approach
The MVP uses a rule-based AI evaluation engine.
Architecture is designed to integrate Google Gemini for future LLM-based
evaluation.
